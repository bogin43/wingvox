Wingvox

Hold Right Option, speak, release. Fully offline voice dictation that runs entirely on your device — nothing leaves it.

Install

curl -fsSL https://raw.githubusercontent.com/bogin43/wingvox/main/bootstrap.sh | bash

Requires an Apple Silicon Mac (M1/M2/M3/M4). Paste this into Terminal — it sets up everything needed and takes a few minutes.

How it works

1
Hold Right Option and speak
2
Audio is transcribed on-device
3
A local LLM cleans it up
4
Pastes wherever your cursor is focused

Before you install

This is a personal project, shared as-is in case it's useful to others, not a polished product with official support. Worth knowing before you run the command above:
  • It requests Microphone, Accessibility, and Input Monitoring permissions — needed to capture audio and paste text into other apps.
  • It installs Homebrew, Xcode Command Line Tools, and Ollama if they're missing on your Mac.
  • Since all transcription and text cleanup happens locally, no audio or text is sent anywhere, meaning it's all private.
Prefer to read the code first? See the source and full README on GitHub.
